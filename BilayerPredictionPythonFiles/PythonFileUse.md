#Bilayer Prediction Instructions (Python 3.9.10 or greater) :
1.	Create a new directory 
2.	Enter the new directory henceforth called `[venv dir]`
3.	Start a python virtual environment with
`python -m venv [venv name]`
4.	Activate virtual environment with
`C:\[venv dir]\[venv name]\Scripts\activate.bat`
5.	Copy all `".py"` files into `C:\[venv dir] `
6.	Install dependencies with 
`C:\[venv dir]\[venv name]\Scripts\pip install -r requirements.txt`
7.	Open “AllAnglesDefinemuBL.py” and assign variables in 
`if __name__ == “__main__”:`
8.	Assign variables in `BilayerPredictionInput.py`
9.	Run `BilayerPredictionRunner.py`
10.	 All output files will be located in `C:\[venv dir]` unless otherwise specified in `BilayerPredictionInput.py`
